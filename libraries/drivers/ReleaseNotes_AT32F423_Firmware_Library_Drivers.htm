Release Notes for

AT32F423 Firmware Library Drivers

V2.0.8-2026/06/02

1.         PWC

l  用于内存屏障的__force_stores()改由__ISB()和__DSB()替代，解决不同编译器下共用问题。

2.         CRM

l  更新crm_reset()，crm_hick_sclk_frequency_select()和crm_hick_divider_select()函数流程，避免切换过程中可能导致的时钟异常。

3.         EXINT

l  新增定义EXINT_TRIGGER_NONE_EDGE，以适应软触发无边沿配置需求。

4.         更新部分注释。

 

V2.0.7-2025/11/24

1.         ADC

l  删除多余的bit位域名spt_bit.cspt18以及多余的寄存器定义csts_bit。

2.         CRM

l  修正bit位域名adc14rst为tmr14rst。

l  删除部分多余的CRM_xxx_PERIPH_LOWPOWER的定义项。

3.         TMR

l  更新tmr_pwm_input_config()、tmr_encoder_mode_config()和tmr_output_channel_config()函数。通道模式配置前关闭通道使能，否则通道模式无法切换。

4.         更新部分注释。

 

V2.0.6-2025/01/20

1.         PWC

l  调整LDO输出档位设定，移除LDO配置项PWC_LDO_OUTPUT_1V0，新增PWC_LDO_OUTPUT_1V1。

 

V2.0.5-2024/12/13

1.         CAN

l  更新发送中止函数can_transmit_cancel()函数的写法，以避免误操作而导致中止命令在各邮箱间互相影响。

2.         USB

l  更新host模式下usb_hch_halt()函数，完善通道halt处理流程。

l  更新host模式下usb_hc_enable()函数，使能对应通道时确保清除其原有状态。

l  更新host模式下usb_host_disable()函数，更新关闭通道流程，以避免误打开其余通道。

l  更新device模式下usb_ept_open()函数，增加同步传输类型的处理，忽略其帧号。

l  更新device模式下usb_ept_close()函数，完善关闭端点流程。

3.         更新部分注释。

1.         更新部分注释。

 

V2.0.3-2024/01/05
